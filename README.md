## Project Name: Flowers Shop

A modern and fully responsive flower shop website designed with an elegant and visually appealing interface. The project delivers a smooth shopping experience across all devices, featuring beautiful floral collections, responsive layouts, and a clean user experience.

### 🌐 Live Demo
🔗 https://menna-salah205.github.io/Flowers-Shop/

### 💻 Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
- **Design Aesthetic:** Elegant floral theme with soft pink tones, modern UI, and fully responsive layouts.
